# Starter React Project

## Dependencies

* React.JS
* TypeScript
* Vite
* SWC
* Bootstrap
* FontAwesome

## Installing Dependencies

```bash
$ make install
```

## Provides

Scaffolding:
* Header
* Collapsible Left Sidebar
* Body
* Footer
* Light, Dark, High-Contrast Themes
* Preferences Context

Controls:
* Buttons (solid, outlined, caret, split buttons, etc.)
* Button Groups
* ListBoxes
* Context Menus
* Modal Dialogs
* Pick Lists
* Panes + Panels

## Running

```bash
$ make serve
```
